Voici les structures exactes que vous pouvez imposer à l'IA d'extraire :

1. Carte Grise (Certificat d'immatriculation)


{
  "document_type": "carte_grise",
  "status": "success",
  "data": {
    "numero_immatriculation": "AB-123-CD",       // Champ A
    "date_premiere_immatriculation": "05/01/1998", // Champ B
    "proprietaire_nom": "DUPONT",                // Champ C.1
    "proprietaire_prenom": "YVES",               // Champ C.1
    "proprietaire_adresse": "27 RUE DES ROITELETS, 59169 FERIN", // Champ C.3
    "marque": "RENAULT",                         // Champ D.1
    "modele": "CLIO",                            // Champ D.3
    "numero_chassis_vin": "VF1BS400523456789"    // Champ E
  }
}

2. Carte Verte (Attestation d'assurance)


{
  "document_type": "carte_verte",
  "status": "success",
  "data": {
    "assurance_compagnie": "MAIF",
    "numero_contrat_police": "F 567 32",

  }
}
